# Shoge

Shoge is a mobile app that you can generate shortcut of links like a Mobile App.
You can generate and custom that Shortcut App choosing a name, enter link to visit, select color or image for the icon, and any browser for open it.
That Shortcut App will generate like a Widget and you can move and put in your smartphone home screen.

---

## Why this project (Shoge) was created?

In GitHub I have 2-3 repositories that are web-tools and can use in the desktop and smartphone. But I needed tap or copy the link of the web-tool to visit it in my smartphone. And I think "Okay. It is a web-tool and I can use it in my smartphone, but is so tedious enter to Whatsapp and tap the link. I want a App of my web-tools..." and obviously is so hard convert a web-tool to app-tool.
But then, I think "What about link shortcut app?" and for that reason I created Shoge.

---

## Good things about this project (Shoge)

- Can generate a Mobile App (Widget) for any link (website, anything).
- Can choose a name of that Shortcut App generated.
- Can select any browser for open it.
- Can choose a color or any image of your gallery for the Shortcut App icon.
